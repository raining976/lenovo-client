## 依赖
- node>=v18.16.0
- npm>=9.5.1

## 安装&使用
1. 使用`git clone`命令将项目克隆到本地
2. 项目根目录中`npm i`安装依赖
3. 运行：`npm run dev`

## 构建发布
`npm run build`打包构建

## 技术栈

- Vue3全家桶
- Vite
- pinia
- axios
- element-plus
- NProgress
